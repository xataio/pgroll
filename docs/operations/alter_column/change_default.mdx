---
title: Change default
description: A change default operation changes the default value of a column.
---

## Structure

```json
{
  "alter_column": {
    "table": "table name",
    "column": "column name",
    "default": "new default value" | null,
    "up": "SQL expression",
    "down": "SQL expression"
  }
}
```
